## Ответы на вопросы
## 1. Что делает fetch?
Метод fetch отправляет HTTP-запросы из браузера и возвращает Promise. Он работает асинхронно, не блокирует интерфейс и используется для загрузки данных (например, цитат и изображений).
## 2. Зачем нужны async/await?
async/await упрощает работу с асинхронным кодом. Он позволяет писать запросы в линейном стиле, без then(), и обрабатывать ошибки через try...catch.
## 3. Как работает localStorage?
localStorage — это постоянное хранилище браузера «ключ-значение». Данные сохраняются между перезагрузками. В работе использовано для хранения выбранной темы.
## 4. Где помог ИИ, а где пришлось разбираться вручную?
ИИ помог: сгенерировать базовую структуру кода, реализовать загрузку данных, исправить ошибки API.
Вручную пришлось разбираться: с каруселью, автопрокруткой, логикой кнопок, отображением статуса «Загрузка…» и применением темы из localStorage.
